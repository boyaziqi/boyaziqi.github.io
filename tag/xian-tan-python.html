    生成器的定义就不冗述了，它可以是函数，也可以是类。可以带参数。接下来分别介绍：

    1：不带参数的生成器（函数）

    def decorate(func):
    def wrapper(*args, **kwargs):
        print("this is a decorate")
        func(*args, **kwargs)
    return wrapper

@decorate
def test():
    print("this is test")

test()

    注意：调用方式是@decorate，后面不跟括号，加上括号就成调用带参数的生成器了。相当于test = decorate(test)。输出结果为：
